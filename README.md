# Striver SDE Sheet - DSA Solutions

This repository contains my solutions to the problems from the popular **Striver SDE Sheet** in Python.

## Repository Structure

- `Arrays/`: Solutions to array-based problems.
  - `set_Matrix_Zeros.py`: Set Matrix Zeroes solution.
  - `pascals_triangle.py`: Pascal's Triangle solution.
  - `next_permutation.py`: Next Permutation solution.
  - `kadanes_algorithm.py`: Kadane's Algorithm (Max Subarray Sum) solution.
  - `sort_0_1_2.py`: Sort an array of 0s, 1s, and 2s solution.
  - `buy_and_sell_stock.py`: Best Time to Buy and Sell Stock solution.
